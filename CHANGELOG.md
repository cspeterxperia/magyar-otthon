# Changelog

## 0.4.0
- Hulladék modul bővítése: 5 külön szenzor entitás (kommunális, szelektív, zöldhulladék, lomtalanítás, fenyőfa).
- Részletes magyar attribútumok minden hulladék entitáshoz.
- Hulladék beállítások frissítése entitás szintű szolgáltatással.
- Integráció automatikus újratöltése opciómódosításkor (újraindítás nélkül).
- Hónapválasztó magyar hónapnevekkel az Options Flow felületen.
- Felhasználói szövegek további magyarosítása (ünnep szenzorok).
- Verzióemelés 0.3.1-ről 0.4.0-ra.
